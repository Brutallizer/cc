Latar Belakang Akademis

Mencegah Pemalsuan Dokumen Menggunakan Kriptografi

Praktik pemalsuan dokumen akademik merugikan integritas institusi pendidikan dan mengacaukan proses rekrutmen di industri. Basis data terpusat (Web2) rentan dimanipulasi oleh insider (orang dalam) dan memiliki titik kegagalan tunggal (Single Point of Failure).

Sebagai solusi revolusioner, CredBlock memisahkan data privat lulusan dengan bukti keabsahannya. Menggunakan Algoritma SHA-256, data dirangkum menjadi nilai Hash yang mustahil dikembalikan ke teks asli. Hash ini kemudian dikunci secara permanen di dalam Smart Contract jaringan Polygon.

Pilar Keamanan Blockchain

Tiga fondasi utama mengapa Ethereal Virtual Machine (EVM) adalah platform sempurna untuk menumpas pemalsuan sertifikat secara global.

Immutability (Kekekalan Data)

Setelah data di-deploy ke dalam block yang valid, mustahil untuk diubah, di-hack, atau dihapus oleh pihak mana pun, mengunci kebenaran validitas akademis.

Transparansi Penyelidikan

Sistem pengecekan tidak berdiri di belakang perizinan tertutup. Pihak HRD merdeka memverifikasi keaslian dokumen secara independen tanpa API server spesifik.

Keamanan Privasi 100%

Sistem Blockchain hanya menyimpan sidik jari digital (Hash). Nama dan transkrip mahasiswa tetap terjaga privasinya di luar server (Off-Chain), anti-doxing.

Arsitektur Eksekusi Terdesentralisasi

Mekanisme visual perjalanan data dari input entitas akademik institusi kampus hingga hasil akhir berupa sertifikasi kriptografi.

1

Entry & Hashing

Kampus memasukkan rekam data kelulusan. Node lokal kami langsung mengkomputasi algoritma kompresi SHA-256 tanpa mencatatkan historinya.

2

Deployment Contract

Melalui dompet web3 (MetaMask), string Hash tersebut ditandatangani secara digital dan dipancarkan ke atas Mainnet Ledger Amoy.

3

Instant Validation

Rekruter global memverifikasi data pelamar secara independen & tanpa latensi melalui portal publik CredBlock yang read-only.
